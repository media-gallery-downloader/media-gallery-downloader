### Prompt 16

**Model:** Claude Opus 4.8

**Context:** Running `media:rename-files` threw `League\Flysystem\CorruptedPathDetected` on a file whose title contains an emoji ZWJ sequence (🏃‍♀️). Root cause: Flysystem rejects any path matching `\p{C}` (control/format chars), and `MediaFilename::sanitizeTitle` kept the zero-width joiner (U+200D). Fix: strip `\p{C}` in sanitizeTitle.
